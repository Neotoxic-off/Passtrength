# Passtrength
Check your password strength 

### Install:
```
git clone https://github.com/Neotoxic-off/Passtrength
```

### Usage:
```
./passtrength <password>
```

### OUTPUT EXAMPLE:

```
Password.......: password

Length.........: 8
Low............: 8
High...........: 0
Number.........: 0
Special........: 0
Occurrence.....: 1

Strength.......: 9
Comment........: Correct password but think to change it quickly

```
